Інтеграції

Зовнішні сервіси та API

LiqPay

Платіжна система для прийому оплати за курси. Підтримка card, Google Pay, Apple Pay.

  • Public / Private ключі (sandbox + production)
  • Webhook для callback-підтверджень оплати
  • Автоматична зміна статусу реєстрації після оплати
  • Рефанди та часткове повернення

Google

Авторизація через Google та One Tap, аналітика, реклама.

OAuth 2.0

Вхід через Google-акаунт. Client ID + Client Secret, redirect URI.

One Tap

Швидка авторизація без перенаправлення. Popup на сторінці входу.

Google Analytics (GA4)

Measurement ID, відстеження конверсій реєстрації та оплати курсів.

reCAPTCHA

Захист форм реєстрації та входу від ботів.

  • reCAPTCHA v3 (invisible) для форм реєстрації та входу
  • Site Key + Secret Key
  • Поріг score для блокування (default 0.5)

Apple

Авторизація через Apple ID (Sign in with Apple).

  • Service ID, Team ID, Key ID, Private Key (.p8)
  • Redirect URI та обробка callback
  • Підтримка приховання email (relay)

Microsoft Clarity

Аналітика поведінки користувачів: теплові карти, записи сесій.

  • Project ID для вбудовування tracking-скрипту
  • Фільтрація адмін-сесій

Facebook

OAuth-авторизація, Facebook Pixel для відстеження конверсій, Open Graph.

  • Facebook Login (OAuth): App ID + App Secret
  • Facebook Pixel: відстеження реєстрацій та оплат
  • Open Graph мета-теги для шерінгу курсів
