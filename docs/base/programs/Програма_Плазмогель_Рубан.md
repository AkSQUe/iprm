## Курс рекомендовано для лікарів

- Дерматологи, косметологи
- Лікарі з досвідом роботи з аутологічною плазмою в естетичній косметології

---

## 📚 Теоретична частина

- Ознайомлення з видами плазми та пробірками, вибір конкретного виду плазми для створення гелю
- Диференціювання колоїдної форми та гелевої
- Причини вибору саме плазмогелю, а не штучного філера. Ефект від гелю / колоїду
- Білки плазми крові
- Підбір обладнання для роботи з термічно обробленою плазмою та алгоритм дій для отримання натурального філера
- Приклади введення препарату та зони роботи
- Конкретні приклади на пацієнтах **до / після**
- Протокол застосування

---

## 🩺 Практична частина

- Перший етап процедури: забір крові, вибір і робота з пробірками, центрифугування, варіння в термостаті, гомогенізація
- Підбір голок для ін'єкціювання, вибір зони для роботи та техніки введення
- Демонстрація виконання протоколів тренером
- Відпрацювання всіх процедур учасниками на практиці під наглядом тренера
- Постановка руки
- Обговорення (питання-відповіді)
